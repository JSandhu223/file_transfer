# File Transfer Program

## Python

Running client:
`python3 client.py <HOSTNAME> <PORT>`

Running server:
`python3 server.py <PORT>`

## C

Running client:

1. Compile with `gcc client.c -o client`
2. Run `./client`

Running server:

1. Compile with `gcc server.c -o server`
2. Run `./server`
